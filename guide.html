1这是什么

首次打开需要联网加载，之后即使断网也可以正常使用——所有功能在本地运行，数据录入与读取均不依赖网络。

Insight · 见微是给老师使用的课后反馈记录工具。它解决的不是“把表格做得更漂亮”，而是把课后最容易流失的观察，变成学期末真正能用的数据。

省时间

每节课后用 3 分钟完成记录。班级、科目、授课内容、学情标签都能快速选择，比在多个系统间复制粘贴更快。

数据积累

每一次课堂表现、作业完成度和个人反馈都会自动沉淀，不再是写完就忘的流水账。

期末报告

学期末可以一键生成每位学生的学科成长报告，把日常反馈、成绩变化和关注标签汇总给家长看。

2新建反馈

最常用的页面。课后选择班级、科目和学生，记录本节课的整体学情与个人表现。

  1. 选择班级与科目：确认本次记录归属到哪个班、哪门学科。
  2. 填写授课内容：可以自由输入，也可以从授课内容库选择。
  3. 勾选班级学情：专注度、互动、作业完成情况等用标签快速完成。
  4. 补充个人反馈：给需要关注的学生记录专注度评分、作业完成度和简短评语。

3历史记录

查看、筛选、修改过去填过的所有反馈。

  1. 筛选：按班级、科目、月份、关键词查找记录。
  2. 查看摘要：记录卡片会显示日期、班级、科目、内容摘要和涉及学生人数。
  3. 回看细节：需要复盘时可以打开完整记录。

4学生档案

把一个学生的反馈、成绩、知识点掌握和自动标签汇总到一张卡片中。

  1. 选择学生：左侧学生列表点击姓名。
  2. 看统计：被记录次数、作业完成率、最近反馈都会自动汇总。
  3. 看画像：雷达图展示各科表现百分比，下方标签帮助判断沟通重点。

5成绩分析

导入月考、模考或单元测成绩，自动生成班级和单科分析。

  1. 录入考试：填写考试名称、日期和满分。
  2. 导入分数：可以手动录入，也可以从 Excel 粘贴。
  3. 查看分析：均分、最高分、最低分、及格率和学生明细自动生成。

7设置与体系切换

切换高考或国际体系，管理班级、学生名单、授课内容库和标签阈值。

  1. 切换体系：设置中选择“高考体系”或“国际考试体系”。
  2. 管理班级和学生：新建班级、批量导入学生名单。
  3. 维护内容库：提前录入常用课程内容和作业，记录时直接选择。

8导出与备份

把数据导出成文件，自己留底，或换电脑继续使用。

  1. 导出 JSON 备份：这是完整数据快照，包含反馈、成绩、设置。
  2. 恢复 JSON 备份：换电脑或浏览器后导入备份即可继续使用。
  3. 导出 Excel / PDF：历史记录、成绩分析和学生报告都可以用于归档或沟通。
建议：每周或每两周做一次 JSON 备份，存到网盘、邮箱或本地硬盘。

9数据安全

真正的风险不是空间不够，而是本地浏览器数据可能被清理。请务必定期备份。

三条会让数据消失的真实风险

  • 清除浏览器缓存或网站数据可能清空本地记录。
  • 换电脑或换浏览器，数据不会自动跟着走。
  • 浏览器异常或系统重装可能造成数据丢失。
怎么防：定期导出 JSON 备份。换电脑时把 JSON 文件导入即可恢复。

10常见问题 FAQ

需要联网吗？
首次通过网址打开时需要联网加载文件。加载完成后，即使断网也可以正常使用所有功能，包括录入数据、查看分析、导出报告。数据保存在本地浏览器中，不涉及任何网络请求。如果使用本地 HTML 文件打开，则完全不需要联网。
数据存在哪里？
数据保存在本地浏览器中。清理浏览器数据、换浏览器或换电脑时不会自动同步，请定期导出 JSON 备份。
可以多个老师共用吗？
同一台电脑、同一个浏览器可以共用一份数据。跨老师协作建议通过定期导出和导入 JSON 备份同步。
能在 iPad 或手机上用吗？
可以。手机适合临时记录反馈；成绩分析、设置和导出报告建议在电脑上操作。
不小心删了一条反馈能恢复吗？
系统内没有回收站。如果有最近的 JSON 备份，可以导入备份恢复。导入前建议先导出当前数据留底。
导入成绩说“未识别到姓名列”怎么办？
检查表格第一行是不是表头，例如“姓名”“学号”“语文”。确认其中一列是中文姓名，并删除多余标题行后再重试。
同一名学生在不同班级出现怎么办？
系统按“班级 + 姓名”识别。换班后的学生在新班级会算作新的学生记录；如需保留完整历史，建议先导出 JSON 备份再处理。
反馈记录会自动备份吗？
不会。请养成定期导出 JSON 的习惯，建议每周至少备份一次，数据量大或使用频繁时可以每天备份。
Excel 导入失败怎么办？
先导出一份系统模板，对照模板检查列名、姓名列和分数列。多余合并单元格、空行、标题行都可能影响识别。
